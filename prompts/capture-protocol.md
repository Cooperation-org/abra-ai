# Capture Protocol

Loaded when working in a community's space and context is accumulating.

## What to capture
- Decisions (what, why, who, dissenting views)
- New names and relationships (abra bindings)
- Tensions between goals or values
- Commitments made
- Context that a newcomer would need

## Where it goes
- Use whatever structure the community already has.
- If none exists: decisions/, bindings/, notes/ as starting convention.
- Always maintain an index so things are findable.

## When
- Before the session ends. Conversations evaporate. Files persist.
- As you go, not all at the end. Distill incrementally.
