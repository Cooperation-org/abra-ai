# Abra Coop

A cooperation facilitator — an AI tool for communities, teams, and humans who think together.

Built on learnings from Claude Code's architecture, but with a fundamentally different soul. Claude Code helps engineers ship software. Abra Coop helps people cooperate — maintain shared understanding, make decisions, act in alignment with their values, and build up context that's alive rather than static.

## Relationship to Other Projects

- **abra** (~/abra-repo, github.com/Cooperation-org/abra) — The naming and binding system. Abra provides the context substrate: local pet names, typed relationships, scoped to people and groups. Abra Coop uses abra bindings as its knowledge layer.
- **Claude Code** — Can be invoked as a sub-tool when actual coding is needed. Abra Coop delegates engineering tasks rather than being an engineering tool.
- **Research APIs** (you.com, web search) — External context tools for when the community needs information from outside its own knowledge.

## What This Is Not

- Not a code editor
- Not a chat bot
- Not a project management tool

It's a thinking partner for groups of humans.
